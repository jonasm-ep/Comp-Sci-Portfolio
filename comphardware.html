    Hardware Description

    Motherboard

    		 The Motherboard is a circuit board that allows the CPU, RAM, and all other computer hardware components to communicate with each other.

    CPU

    		 The CPU (Central Processing Unit) is a processor that sends signals to control the other parts of the computer. It is where all the calculations in a computer takes place.

    RAM

    		 The RAM (Random-access Memory) is a form of computer data storage. The RAM stores data that will be used later.

    Hard Drive

    		 The hard drive is the computer's main storage media device that permanently stores all data on the computer.

    How do the different components in a computer communicate with one another?

    		 Computer components are connected to each other via thin connections known as 'bus' on the motherboard which is how they communicate.

    Keyboard

    		 The keyboard is a board with a bunch of keys which is used to type and enter data onto a computer.

    Mouse

    		 The mouse is the device that you use to move the cursor on a computer to click links, buttons etc. .

    Monitor

    		 The device with a screen that is used to display the computer's actions.
